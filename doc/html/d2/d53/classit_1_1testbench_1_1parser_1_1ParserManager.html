Detailed Description

ParserManager component provides the capabilities to safely load, parse and validate a configuration file.

Internally, an instance of ParserManager acts an FSM (Finite State Machine) for which the processing flow is well defined: O = f(I, S), its implementation is done using a Moore machine model.
