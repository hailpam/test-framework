Detailed Description

template<class T>
class it::testbench::rte::SynchronizedQueue< T >

Synchronized Queue is a template class allowing a synchronized access to its internal state.

FIFO (First-In, First-Out) policy provided by the interface.

It wraps POSIX pthread synchronization mechanisms.
